Projeto Curie

Este site representa o último projeto dos exercícios da Labenu. O Portfólio visa apresentar ao menos 3 reposítórios de projetos realizados durante o curso. No meu caso, resolvi apresentar não só 3 mas todos os projetos que eu fiz e tiveram um resultado satisfatório. Segue abaixo o link de cada um deles.
AmbulnzLLC
App de cardápio para pizzaria, com menu para escolha de sabores e realização de pedidos. Projeto fullstack da rodada de casases com API e documentação. Link do respositório: https://github.com/Gazua300/AmbulnzLLC
E-commerce
App que simula um pequeno marketplace, com filtragem de produtos por valor e carrinho. Link do respositório: https://github.com/Gazua300/E-commerce
Pokedex
Lista Pokedex de pokemons usando a API pública https://pokeapi.co/. Link do respositório: https://github.com/Gazua300/Pokedex
Labeninja
Marketplace para cadastro e contratação de serviços. Link do respositório: https://github.com/Gazua300/labeninja
Amaro Fashion
Projeto backend. API com endpoints para produtos de vestimenta. Link do respositório: https://github.com/Gazua300/Amaro
Labebank
Projeto backend. API para app de conta bancária com acesso a várias transações, tais como saldo, extrato, pagamentos, transferências etc. Link do respositório: https://github.com/Gazua300/labebank
