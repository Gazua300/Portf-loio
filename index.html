Projeto Curie

Este site representa o último projeto dos exercícios da Labenu. O Portfólio visa apresentar ao menos três projetos realizados durante o curso. No meu caso, resolvi apresentar não só três mas os projetos que fiz e tiveram um resultado satisfatório. Segue abaixo os links do deploy e API de cada um deles.
E-commerce (frontend)
App que simula um pequeno marketplace, com filtragem de por valor e carrinho para adicionar os produtos.

Respositório: https://github.com/Gazua300/E-commerce
Loja de Serviços(fullstack)
Projeto fullstack de app para cadastramento e divulgação de serviços diversos: https://github.com/Gazua300/achei_servicos-server
Fakebank (fullstack)
Aplicativo para operações bancárias. Simula as mais básicas operações como saldo, extrato,pagamentos e transfereências.

Repositório: https://github.com/Gazua300/labebank
Restrições de acesso (backend)
API com endpoints para guardar informações pessoais, sobre educação e experiências profissionais dos usuários. As informações tem acesso restrito através de um token que é gerado ao criar a conta e quando é realizado o login.

Repositório: https://github.com/Gazua300/Restricoes-de-acesso
Ifuture (frontend)
Aplicativo que simula serviços de pedido e entrega de comidas, lanches e bebidas. O usuário tem à disposição uma lista com várias lojas do ramo de alimentos e bebidas, cada uma com seu cardápio para realizar os pedidos.

Repositório: https://github.com/Gazua300/labefood
